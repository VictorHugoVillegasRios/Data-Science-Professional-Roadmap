# 🧠 Rutas de Formación – Ciencia de Datos

Esta sección reúne los programas profesionales más reconocidos a nivel internacional para iniciar o consolidar una carrera en **Ciencia de Datos**.

Se priorizan certificaciones y programas:

✔ Con reconocimiento global  
✔ Con enfoque práctico  
✔ Orientados al mercado laboral  
✔ Con herramientas actuales (Python, R, SQL, Machine Learning, Big Data)  

---

## 🥇 IBM Data Science Professional Certificate
- **Institución:** IBM  
- **Plataforma:** Coursera  
- **Nivel:** Inicial – Intermedio  
- **Duración estimada:** 6 a 9 meses  

**🔎 Enfoque**  
Programa integral que cubre desde fundamentos de ciencia de datos hasta aplicaciones prácticas en Python y Machine Learning.  

**🎯 Competencias desarrolladas**  
- Python para ciencia de datos  
- Análisis estadístico  
- Visualización de datos  
- Machine Learning básico  
- Creación de proyectos aplicados  

🔗 [Enlace oficial](https://www.coursera.org/professional-certificates/ibm-data-science)

---

## 🥈 Google Advanced Data Analytics Professional Certificate
- **Institución:** Google

## 🥉 MITx MicroMasters in Statistics and Data Science

- **Plataforma:** edX

- Machine Learning avanzado  
- Modelado predictivo
